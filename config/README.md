# NeuroGraph Configuration

Центральное место для всех конфигурационных файлов проекта.

## Структура

```
config/
├── README.md              # Этот файл
├── project.env            # Переменные окружения проекта
├── python.env             # Python-специфичные настройки
├── rust.env               # Rust-специфичные настройки
└── versions.env           # Версии зависимостей
```

## Использование

Все скрипты автоматически загружают конфигурацию из `.config.sh` в корне проекта,
который в свою очередь использует файлы из этой директории.

## Переопределение настроек

Для локальных переопределений создайте файл `.env.local` в корне проекта.
