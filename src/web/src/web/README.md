# NeuroGraph Web Dashboard

Modern React SPA for monitoring and managing the NeuroGraph OS cognitive architecture system.

## Quick Start

### Prerequisites

- Node.js 18+ and npm
- NeuroGraph backend running on `localhost:8000`

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The dashboard will be available at **http://localhost:5173**

### Production Build

```bash
# Build for production
npm run build

# Preview production build
npm run preview
```
